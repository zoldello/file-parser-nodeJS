File Parser in node.js
==================================================

How to set up
--------------------------------------
1. Get Node.js
2. Get dependencies by running:
```bash
 npm insall
```

How to Test and Link
--------------------------------------
Run:
```bash
 gulp
```

How to use
--------------------------------------
Run:
```bash
node index someFile.txt
```

where someFile.txt is a comma, space or piple separated file
